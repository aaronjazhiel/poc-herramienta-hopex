Paso 1 — Validación de Existencia

Verificar si la aplicación o sistema ya existe en HOPEX antes de crear

Se usa filtro name_contains para búsqueda parcial
POST https://pprod-gnp.hopexcloud.com/HOPEXGraphQL/api/ITPM 
query {
  application(filter: { name_contains: "Cotizador Seguros Vivienda" }) {
    id
    name
    creationDate
    creator { id name }
  }
}
